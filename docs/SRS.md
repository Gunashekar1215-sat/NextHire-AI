# Software Requirements Specification (SRS)

# NextHire AI

---

# 1. Introduction

## 1.1 Purpose

NextHire AI is an AI-powered career assistant platform designed to help students improve their resumes, prepare for interviews, identify skill gaps, and build personalized career roadmaps.

The platform combines multiple career services into one intelligent system.

---

## 1.2 Objectives

The primary objectives are:

- Resume Analysis
- ATS Score Prediction
- Resume Improvement Suggestions
- Interview Preparation
- Skill Gap Analysis
- Career Roadmap Generation
- Job Recommendation
- Portfolio Analysis

---

# 2. Problem Statement

Students generally use different websites for:

- Resume Building
- ATS Checking
- Interview Preparation
- Career Planning
- Job Searching

This wastes time and provides inconsistent guidance.

---

# 3. Proposed Solution

NextHire AI provides all these services inside a single platform using Artificial Intelligence.

---

# 4. Target Users

- Students
- Freshers
- Job Seekers
- Career Switchers

---

# 5. Functional Requirements

The system shall allow users to:

- Register/Login
- Upload Resume
- Analyze Resume
- Generate ATS Score
- View Skill Gap
- Generate Interview Questions
- View Career Roadmap
- View Job Recommendations
- Manage Profile

---

# 6. Non Functional Requirements

- Secure
- Fast
- Responsive
- User Friendly
- Scalable
- Reliable
