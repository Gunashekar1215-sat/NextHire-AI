# NextHire AI - Project Plan

## Project Name
NextHire AI

## Project Vision
NextHire AI is an AI-powered career assistant designed to help students improve their resumes, prepare for interviews, identify skill gaps, and generate personalized career roadmaps.

## Target Users
- Students
- Freshers
- Job Seekers

## Core Features

1. AI Resume Analyzer
2. ATS Score Checker
3. Resume Improvement Suggestions
4. AI Interview Question Generator
5. Personalized Career Roadmap
6. Skill Gap Analysis
7. Job Recommendations
8. Portfolio Analysis

# Software Requirements Specification (SRS)

## 1. Project Overview

### Project Name
NextHire AI

### Project Type
AI-Powered Career Assistant Platform

### Problem Statement
Students struggle with resume building, interview preparation, career planning, and identifying missing skills. Existing platforms solve only one or two of these problems, forcing users to switch between multiple websites.

### Proposed Solution
NextHire AI combines Resume Analysis, ATS Score, Skill Gap Analysis, AI Interview Preparation, Career Roadmap, Job Recommendations, and Portfolio Analysis into a single platform.
